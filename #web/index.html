cchlib

Here is the last cchlib JAVA Library, more informations available at https://code.google.com/p/cchlib/.

Module nameDescriptionLast documentationjarsComments

cchlib-i18n-deprecated Previous version of cchlib-i18n. jars (Java 1.6 required, unmaintained, no further documentation)
cchlib-io Provide basic io related features. javadoc jars (Java 1.6 required)
cchlib-j2ee Provide server based features. javadoc jars (Java 1.6 required, unmaintained)
cchlib-j2ee-deprecated Contain deprecated classes of cchlib-j2ee library. jars (Java 1.6 required, unmaintained, no further documentation)
cchlib-jdbf Provide tools to read and write DBF files. javadoc jars (Java 1.6 required)
cchlib-net Network API javadoc jars (Java 1.6 required)
cchlib-nio Provide feature using new I/O javadoc jars (Java 1.7 required)
cchlib-sql Provide SQL related features. javadoc jars (Java 1.6 required)
cchlib-swing Some swing extensions javadoc jars (Java 1.7 required)
cchlib-swing-deprecated Contain deprecated classes of cchlib-swing library. jars (Java 1.6 required)
